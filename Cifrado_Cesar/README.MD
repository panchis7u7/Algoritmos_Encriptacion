## Algoritmo de cifrado de Cesar. 
### Carlos Sebastian Madrigal Rodriguez.

## Compilacion: 

### Make:
```bash
make
```

### Usando gcc:
```bash
gcc -o cesar cesar.c
```

### Ejemplos de ejecucion: 
```bash
./cesar [numero de rotaciones] --crypt [Texto a encriptar]
./cesar [numero de rotaciones] --decrypt [Texto a desencriptar]
```
